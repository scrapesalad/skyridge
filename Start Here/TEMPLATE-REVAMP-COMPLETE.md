## Next Steps for Developers

When using this template:

1. **Start with `index.html`**
   - Follow the STEP comments
   - Replace all `PUT YOUR...HERE` placeholders
   - Update images, links, and content

2. **Use Template Files**
   - `pages/_TEMPLATE-BASE.html` - For standard pages
   - `pages/_TEMPLATE-SERVICE.html` - For service pages
   - `pages/_TEMPLATE-CITY.html` - For location pages

3. **Follow the Guide**
   - See `Start Here/KING-TUT-TEMPLATE-GUIDE.md` for detailed instructions
   - Reference `Start Here/METADATA-PLACEHOLDERS-REFERENCE.md` for SEO help

4. **Update Configuration**
   - Edit `site.config.json` with your site details
   - Update logo, colors, contact info, etc.
