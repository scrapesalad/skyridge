# King Tut Site Generator Templates

This directory contains the template files used by the King Tut site generator.

## Template Structure

Each template folder should contain:
- `index.html` - Main HTML file with token placeholders (e.g., `{{BUSINESS_NAME}}`)
- `css/` - Stylesheet files
  - `styles.css` - Main stylesheet
  - `design-tokens-override.css` - Color scheme (auto-generated by generator)
  - `avada-enhancements.css` - Additional enhancements
- `images/` - Image assets (optional, will be copied if present)
- Any other static assets needed

## Available Templates

1. **business-basic** - Standard business website template
2. **business-premium** - Enhanced business template with additional features
3. **portfolio** - Portfolio/showcase template
4. **agency** - Agency/creative template
5. **onepage** - Single-page landing template

## Token Placeholders

The generator replaces these tokens in all HTML, CSS, JS, and text files:

- `{{BUSINESS_NAME}}` - Business/company name
- `{{SITE_NAME}}` - Website name (defaults to business name)
- `{{SERVICE_TYPE}}` - Primary service type
- `{{PRIMARY_SERVICE}}` - Same as SERVICE_TYPE
- `{{SERVICE_AREA}}` - Service area/region
- `{{REGION}}` - Same as SERVICE_AREA
- `{{YEAR_STARTED}}` - Year business started
- `{{CURRENT_YEAR}}` - Current year
- `{{CITY}}` - Primary city
- `{{HOME_PAGE_DESCRIPTION}}` - SEO description
- `{{KEYWORDS}}` - SEO keywords
- `{{WEBSITE_URL}}` - Website URL
- `{{BUSINESS_DESCRIPTION}}` - Business description
- `{{PHONE_NUMBER}}` - Contact phone
- `{{EMAIL}}` - Contact email
- `{{STREET_ADDRESS}}` - Business address
- `{{POSTAL_CODE}}` - ZIP/postal code
- `{{COUNTRY}}` - Country
- `{{FACEBOOK_URL}}` - Facebook page URL
- `{{INSTAGRAM_URL}}` - Instagram profile URL
- `{{LINKEDIN_URL}}` - LinkedIn company URL
- `{{GOOGLE_ANALYTICS_ID}}` - Google Analytics ID
- `{{PRIMARY_COLOR}}` - Primary brand color (hex)
- `{{TARGET_AUDIENCE}}` - Target audience description

## Adding a New Template

1. Create a new folder in `templates/`
2. Add your `index.html` with token placeholders
3. Add CSS files in `css/` directory
4. Add any images or assets
5. Update `scripts/create-site.js` to include your template in the `TEMPLATES` array
